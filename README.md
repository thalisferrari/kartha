# Kartha - Nossa História de Amor

Timeline interativa contando a história do casal, construída com Next.js e hospedada no GitHub Pages.

## Stack

- **Next.js 15** (App Router, TypeScript, Static Export)
- **Tailwind CSS** (Tema preto + azul)
- **Framer Motion** (Animações)
- **Lucide React** (Ícones)
- **Moment.js** (Datas em PT-BR)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Os arquivos estáticos são gerados na pasta `out/`.

## Deploy

O deploy é feito automaticamente via GitHub Actions ao fazer push para a branch `main`. O site é publicado no GitHub Pages.

## Personalização

Consulte o [Guia de Conteúdo](docs/guia-de-conteudo.md) para instruções de como editar o conteúdo da timeline.
